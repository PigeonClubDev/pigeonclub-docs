# 加入

## 基本要求

> **我们不欢迎下列情形的人提交申请**，如果申请未通过，请确认是否存在下列情形。

* 患有精神疾病、心理疾病或障碍的。
* 有任何违法或严重违纪行为史的。
* 心理或行为不符合主旋律，传播反人类、反科学、反辩证、反社会思想的。
* 对社团管理员态度恶劣的。
* 在社团或YukiGeek有被开除记录的。

> **以下情形会经过多重考核后通过**

* 对所申请专业或领域了解过少。
* 追星或盲目崇拜任一实体。
* 日常娱乐以快节奏内容为主。
* 正处于青春期恋爱中。

> **填写申请表时，请注意以下规范。**

1. 填写真实信息并对填写的所有内容负责。
2. 不允许使用QQ小号，请使用常用的QQ。
3. 无法确认QQ邮箱是否开启时，请填写其他邮箱。
4. 自我描述请尽量详细全面。

## 其他要求

对应的特殊要求，可以在下一页“[角色与岗位](role.md)”查阅。
