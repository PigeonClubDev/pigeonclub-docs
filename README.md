# PigeonClub - 鸽社主页

## 欢迎

**本页面持续完善中**

欢迎来到PigeonClub！

你已经有所耳闻，或是头一回得知我们社团的活动。我们秉持“度过一段充实而有意义的高中生涯”或是“独善其身而不失陪伴”的理念，维持着亚文化爱好者在南康区本地的一分三亩地。

如果你有着独到的想法，欢迎你的入社申请！

## 社团性质

我们独立于任何形式的校官方组织，是隶属于YukiGeek工作室。最终受YukiGeek工作室的成员管辖的同人社团。

这意味着我们与校团委有关的任何考核无关，因此，如果你出于此目的加入我们，请放弃这样的想法。当然，我们的活动一般不与校团委的大部分活动冲突，二者可以兼得。

社团主要从事文字冒险游戏与小型2D游戏的开发。广泛招收乐团、文案、美术等相关人员。当然，对我们而言最重要的并非专业能力，而是应对某项专业能得出自己独到的见解并有着较强的自学能力。

提出申请与加入社团是自由的，退出同样是自由的。当然，你可能需要经过一段时间的考核才能进入社团。

## 活动

我们仅在江西省赣州市南康区市区活动。

社团不定时召开线下见面，时间上冲突并存在特殊原因的同学允许通过线上的方式参与。

PigeonClub Online采取管理选拔邀请制，并与ColorSail Minecraft服务器深度合作。欲加入PCOL，请先加入ColorSail服务器。你可以在下文的友情链接找到他们。

对于参与活动不积极的成员，社团管理员视情况开除其成员资格。
