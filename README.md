# 欢迎加入PigeonClub大家庭！

## 这是什么

这儿是PigeonClub社团文档中心，包括社团及一系列周边设施使用、管理、操作规范与说明。

我们希望无论新加入的成员在任何专业知识上有任何的欠缺，都能熟练使用PigeonClub的资源及其周边服务。

希望各位成员能够从文档中学会许多新的知识，并度过一个美好的学习时光。

## 文档的适用群体

无论你是最近才通过加入审核，亦或是另一种含义的“萌新”，这篇文档将会详细解释社团的组织结构及相关用语解释。

管理层会不定时更新文档，以满足社团未来阶段的需求。
